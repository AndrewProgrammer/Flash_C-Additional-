Переменная +
Правила именования переменных +
Выражение - Любое выражение языка состоит из операндов (переменных, констант и др.), соединенных знаками операций(либо операторами),и которые всегда имеют результатирующее значение. 
Преобразование типов - это операция по получению значения одного типа данных из значения другого типа данных.Оно бывает:Явным,неявным,расширяющим,сужающим.(Как производиться преобразование типов данных в выражении : будет происходит преобразование до найбольшего типа данных, так как компилятор стремиться привести их к одному типу данных.)
Что такое тип данных+
Массив+
Обращение к элементам массива - осуществляется при помощи индексов,и индексация массива начинается с 0.
Что такое индекс - номер элемента массива,указывает на конкретный элемент массива.
Почему индексация начинается с 0 -потому что смещение адреса первого элемента массива от начала этого массива равно нулю
Что такое функция - это специальный инструментарий кода, который позволяет повторяющейся фрагмент кода вынести за тело программы.	
Что такое перегрузка функций - это несколько функций( 2 или больше) с одинаковыми именами, но различными параметрами.	
Встраивание функций - это код функции непосредственно в коде программы по месту вызова, что дает выйграшь во времени, компилятору не приходиться создавать функцию в памяти.Используется с рекомендацией inline
Указатель - это переменная хранящая в себе адрес другой переменной.
Адрес - это порядковый номер байта, начиная с которого переменная хранится в памяти
Процедурное программирование - программирование на императивном языке, при котором последовательно выполняемые операторы можно собрать в подпрограммы, то есть более крупные целостные единицы кода, с помощью механизмов самого языка.
Арифметика указателей - это способ вычисления адреса какого-либо объекта при помощи арифметических операций над указателями, а также использование указателей в операциях сравнения.Они бывают:Присваивание(*),Взятие адреса(&),Косвенная адресация,Преобразование типа,Определение размера, Сравнение, Индексация,Увеление/Уменьшение,сложение,вычитание.
Динмаическая память - память компьютера, необходима нам тогда, когда в стеке уже нет места для поддержки нашего кода.амять резервируется не на этапе компиляции а на этапе выполнения программы. 
Операция разыменования - это означает что вместо переменной указателя которая хранит в себе адрес мы отображаем значение переменной по которому ссылается указатель.
Зачем нужны указатели - для работы с динамической памятью компьюетра.
Ссылки - псевдоним перемоной которая при использовании, работает уже со значением, так как имеет автоматическую операцию разыменования.
Разница ссылки и указателя - тем что указатель это переменная которая ссылается на адрес другой переменной(выражения), и для работы со значением переменой(выражения) которое хранит адрес, нужно использовать операцию разыменования, а ссылка при использовании автоматически разыменовывается.
Структура зачем нужна - для создания определенного списка элементов(выражений) с разными типами данных, с которомы мы возможно будем работать на протяжении всей программы.
Что такое файл. - это определенная последовательность бит имеющих общее имя,уникальное имя.
Режимы открытия файла - r - только чтение. Запись запрещена. Файл должен существовать.
			w - только запись. Чтение запрещено. Создает пустой файл. Если файл с таким именем существует,
				то fopen его очистит до нулевой длины. (Всё содержимое файла будет потеряно, и он будет пуст)
			a - запись в конец файла. Если файла не существует, он будет создан. Если файл существует, fopen откроет его
				и ничего очищать не будет. Последующая запись будет дописана в конец файла.
			
			
			r+ - открытие для чтения и записи. Файл должен существовать. Чтение начинается с начала файла.
			w+ - открытие для чтения и записи. Создает пустой файл. Если файл с таким именем существует,
				то fopen его очистит до нулевой длины. (Всё содержимое файла будет потеряно, и он будет пуст)
			a+ - открытие для чтения и записи. Если файла не существует, он будет создан. Если файл существует, fopen откроет его
				и ничего очищать не будет. Последующая запись будет дописана в конец файла.
Модификаторы - ключевые слова signed, unsigned.
Спецификатор типа - ключевое слово char или int, определяющее тип объявляемой переменной.
Идентификатор - имя переменной.
unsigned,signed char - 1 байт.
unsigned,signed int,shor - 2 байта.
unsigned,signed long - 4 байта.
unsigned.signed long long - 8 байт

Элементы одинаковые но по разному инициализируются.
List Box - предназначен для того чтобы отоброжать список элементов
BOOL OnInitDialog(HWND window, HWND /*focus*/, LPARAM /*lParam*/) - нужна для заполнения информации.
Сначала запоминаются дескрипторы.
LPARAM Лист бокса - можно передовать строки.
LB_SETCOLUMNWIDTH - устанавливает ширину колонки.
LB_ADDSTRING - добавляются строки во 2 лист бокс.
IDC_BUTTON_ADD - срабатывает, если мы нажмем на кнопку AddPeson
LB_FINDSTRINGEXACT - ищет строку в лист боксе, переданную ей.WPARAM(-1) - поиск производиться во все лист боксе, если мы хотим с определенного индекса то от того индекса до конца.
LB_ERR - (-1)
IDC_BUTTON_DELETE - удаление элемента
LB_GETCURSEL - правильный выбраный элемент с одним элементом.
LB_GETTEXTLEN - длина текста самого list box.
WM_GETTEXTLENGHT - ассоцируемое ко всему окну, для разных окон.
LB_GETTEXT - наш выбранный элемент
Память динамическую выделяем саму.
А уже в функции передаем указатель.
IDC_BUTTON_SEARCH - поиск нашего элемента 
LB_SELECTSTRING - пытается выделить подходящую строку
IDC_BUTTON_CLEAR - кнопку удаления текста.
LB_RESETCONTENT - полностью убивает все содержимое List Box
Очищать List Box нужно при добавлении, то есть все убить и добавить по новой.
IDC_BUTTON_GET_SELECTED - позволяет несколько элементов выдела
динамический массив нам нужен для управлением целым списком выбранных элементов.
Обработка выделения:
LBN_SELCHANGE - то что мы выбрали, конкретный код.
IDC_LIST1 - айдишник List Box
index != LB_ERR - если убралось выделение.
LB_GETSELITEMS - наши элементы
------------------------------------------
Выпадающий список (Combo Box):
LB_SETITEMDATA - добавляет адрес элемента(каждый разный) для list
CB_SETITEMDATA - добавляет адрес элемента(каждый разный) для combo
LB_GETITEMDATA - получение нашего адреса для обращения

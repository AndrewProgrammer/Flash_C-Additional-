Edit_Control - WinApi
окно в который можно писать текст.
Для создания техта с окном, можно использовать ES_LEFT,ES_RIGHT,ES_CENTER,ES_LOWERCASE,ES_UPPERCASE,ES_NOHIDESEL(если элемент потеряет фокус, то все равно оно будет работать)ES_NUMBER - только цифри,ES_PASSWORD - будут выводится кружечки,ES_READONLY - нельзя печатать в него, 2 стиля которые работают вместе ES_MULTILINE - элемент многострочный, каждый раз пишет заново,ES_WANTRETURN - при нажатии на ENTER переходит на новую строку // Стили
------------------------------------------------------
Функции:
SetWindowLong() - наше окно, параметр который мы хотим изменить, и на что хотим поменять, возвращает указатель на прошлое окно
GetWindowLong();
// Переопределение оконной процедуры текстового поля. Другая обработка
            g_originalProcedure = reinterpret_cast<WNDPROC>(
                SetWindowLong(g_edit, GWL_WNDPROC, reinterpret_cast<LONG>(EditProcedure))
            );
SetWindowLong - изменяет поля для конкретного окна
SetClassLong - изменяет для всего класса
SetClassLongPtr -изменяет для всего класса
WM_POSITION - меняет позициую в данном случаи сообщение курсора WM_APP - начало значений свободного диапозона
EM_GETSEL - получение выделения, полного диапозона, индекс начала и индекс конца будут совпадать, индекс начала будет описовать символы которые будут предыдущие, а индекс конца это символы которые позже в строке
if (IsClipboardFormatAvailable(CF_TEXT)) - буфер обмена, передаем текст
 EM_CANUNDO - отвечает за можно ли сделать откат?
WM_GETTEXTLENGTH - длина текста.
EM_LINEFROMCHAR - по индексу опеределяет, какая это строка(номер строки) от всего нашего текста
static_cast<WPARAM>(-1) - индекс где находится курсор ввода
EM_LINEINDEX - по индексу определяет, индекс относительно всей нашей строки

if (HIWORD(wParam) == EN_CHANGE) - HIWORD - notification code
EN_CHANGE - если в edit поменялся текст, значит текст поменялся
